Openshift Legal Contract Documents Analyzer: 

An AI Tools that help Businesses, Companies etc. to Simplify, Analyze, Summarize and  Translate Legal Contract Documents leveraging Openshift Technologies and ChatGPT AI



This Application is written with PHP, Mysql Database, Curl, Ajax, Jquery, Bootstraps etc...



1.)You will need to install Xampp Server. After installation, ensure that PHP and Mysql Server has been started and Running from Xampp Control Panel.


2.) copy main application folder clarifai_document_analyzer which contains the backend Code to htdocs of the Xammp server. Eg. C:\xampp\htdocs\clarifai_document_analyzer.


3.) open phpmyadmin interface  of the Xammpp Server Eg at http://localhost/phpmyadmin/ to create database name Eg. clarifai_db.

4.) Locate file clarifai_db.SQL to import users tables to Mysql Database you created above.

5.) Locate  and Edit data6rst.php file to reflect mysql database credentials.

6.) locate and Edit Settings.php to update All Dropbox Hellosign API and Clarifai Access Token and other Parameters where appropriates. 




It is time to test the app.


7.) Callup Browser at http://localhost/clarifai_document_analyzer/index.php and the application will start running. Thanks.. 




API Used


1.) Dropbox Hellosign API ( https://www.hellosign.com/ )

2.) Clarifai ( https://www.clarifai.com/ )

3.)PDF parser library (https://github.com/smalot/pdfparser)
